HTML

HTML é uma linguagem de Marcação de HiperTexto, tem como função e objetivo de:

  • ESTRUTURAR A PÁGINA WEB
  • INSERÇÃO DE CONTEÚDOS COMO IMAGENS E VÍDEOS POR MEIO DE HIPERTEXTOS.
  • ULTILIZA MARCADORES (TAGS) E ATRUBUTOS QUE DEFINEM COMO O CONTEÚDO DEVE SER APRESENTADO NO NAVEGADOR.

HTML - HyperText Markup Language - Linguagem de marcação de HiperTexto criado pelo físico britânico, Tim Berners Lee, é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos por meio dos hipertextos.
